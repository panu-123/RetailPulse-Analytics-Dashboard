# 📊 RetailPulse Analytics Dashboard

An interactive Business Intelligence dashboard built using **Qlik Sense** to analyze retail sales, profitability, customer behavior, and regional performance.

## 🚀 Features

- Executive KPI Dashboard
- Sales Trend Analysis
- Profit Analysis
- Customer Insights
- Interactive Filters
- Top Products Analysis
